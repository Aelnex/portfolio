📁 โครงสร้างโปรเจกต์
========================

portfolio/
├── 📄 package.json              # การตั้งค่าโปรเจกต์และ dependencies
├── 📄 tailwind.config.js        # การตั้งค่า TailwindCSS
├── 📄 postcss.config.js         # การตั้งค่า PostCSS
├── 📄 .gitignore                # ไฟล์ที่ไม่ต้อง push ขึ้น GitHub
├── 📄 README.md                 # คู่มือใช้งานฉบับเต็ม
├── 📄 DEPLOY_GUIDE.md           # คู่มือ deploy แบบย่อ
│
├── 📁 public/                   # ไฟล์สาธารณะ
│   └── 📄 index.html            # HTML template หลัก
│
└── 📁 src/                      # โค้ดหลัก
    ├── 📄 index.js              # Entry point
    ├── 📄 index.css             # Global CSS + Tailwind
    └── 📄 Portfolio.jsx         # Component หลักของเว็บ

========================

✅ ไฟล์ที่ต้องแก้ไขก่อน Deploy:
1. src/Portfolio.jsx - แก้ไขข้อมูลส่วนตัว, โปรเจกต์, skills
2. package.json - แก้ homepage ให้ตรงกับ GitHub username

📦 Dependencies ที่ติดตั้ง:
- React 18
- TailwindCSS
- Lucide React (icons)
- gh-pages (สำหรับ deploy)

🚀 คำสั่งที่ใช้บ่อย:
- npm install        → ติดตั้ง dependencies
- npm start          → รันในเครื่อง
- npm run build      → Build สำหรับ production
- npm run deploy     → Deploy ไป GitHub Pages
